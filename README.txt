# Ethel Jena | Safety and Security Manager

A professional one-page profile site for Ethel Jena, built using the **Hyperspace** template by [HTML5 UP](https://html5up.net). This responsive site highlights Ethel’s leadership journey in safety, security, and cross-cultural engagement through Peace Corps South Africa.

### Features
- Modern, responsive layout
- Professional “About” section with official appointment and evaluation visuals
- Curated highlight gallery with links to social features and service milestones
- Integrated contact card for public speaking or outreach inquiries

### Project Structure
- `index.html` – Main profile site
- `images/` – Profile images, announcement and evaluation screenshots
- `assets/css/` – Site styles (from Hyperspace)
- `assets/js/` – Scroll animations and utility scripts
